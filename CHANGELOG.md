
# v1.3.1 (2026-01-29) Offline Map Update

### 🚀 新增功能
*   **離線地圖支援**:
    *   實作「地圖快取預載器 (Map Preloader)」。

# v1.3.0 (2026-01-29)

### 🚀 新增功能
*   **即時導航系統**:
    *   新增「開始導航」按鈕，可利用 GPS 追蹤目前位置。
    *   新增「定位我」按鈕，快速鎖定視角。
    *   地圖上以**藍色脈衝圓點**顯示位置，並以**藍色虛線**繪製行走軌跡，與規劃路徑區隔。
    *   顯示 GPS 精確度與海拔資訊。
*   **版本控制系統**:
    *   介面標題顯示版本號 (v1.3.0)。
    *   匯出 GPX 檔案時，metadata 自動寫入版本資訊 (`HikingMapTool/1.3.0`)。

### 🐛 修正與優化
*   修正 `.gpx` 匯出邏輯中的語法錯誤。
*   優化工具列排列。
